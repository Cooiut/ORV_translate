外传622章 Ep14. 狩猎(1)

刘众赫毫不犹豫地大步领先，也没有说要去哪里。

李贤成、金南云、李智慧紧随其后，各自只差一两步。

我们保持一定的距离追赶着刘众赫一行。

就那样追了多久。

走在我旁边的京世仁悄悄地发出了担心的声音。

"可以继续跟着吗？"

"我不是说你不用跟上来吗？"

我们没有选择属于"忠武路站"，而是选择了"流浪者"。

这是一个冒险的决定，所以在做出这个选择之前，我提前征求了京世仁、丹数叔叔和郑熙媛的意见。

-虽然眼下最安全的是忠武路，但真正陷入危机时能救我的应该是仁浩吧。

京世仁。

-可以一起边走边找智允吗？

丹数叔叔。

-我想如果我留下你一个人，你会死在某个地方。

最后是郑熙媛。

到了这个程度，我想我们前世不会也算是队友吧。

"仁浩，谢谢你送的新剑。现在道谢会不会太晚了。"

"你喜欢吗？"

郑熙媛在空中挥舞着三日月宗近。

"非常喜欢。"

即使这样，她的目光仍然没有离开刘众赫的背影。

看到"刘众赫"的武力后依然能燃烧斗志，不愧是郑熙媛。

除了回归者或归来者等特殊情况外，普通人中最强的恐怕就是郑熙媛了。

离开忠武路站完全登上地面后，刘众赫停下了脚步。

他漫不经心地环顾了一圈已是废墟的街道，然后拦住了同伴。

"等等。"

刘众赫把长枪插在地上，他轮流看了看西边和北边的街道，陷入了沉思。

这个地点的西边和北边。

虽然隐约想起了一些东西，但信息不足，无法断言。

如果能有"灭活法"就好了。

现在我拥有的只有读者的评论和偶尔想起的笔记。

在远处踢着地板的金南云看到了我们。

在笑嘻嘻的家伙正要往这边走的时候，李智慧发出了冰冷的声音。

"别没事找事，如果你不想被我揍的话。"

"嗯？好吧。"

代替犹豫不决的金南云向我们走来的，是刘众赫一行人中最温和善良的男人。

"你好，没有受伤吧？"

"啊，是的，谢谢。"

"对不起。我们队长话比较少。"

"没什么，我们只是自己跟上来。"

眼神憨直的李贤成，微笑着露出了强壮的胸肌。

看着他淳朴的表情，我再次意识到自己已经真正进入了全读视中。

真的是'钢铁剑帝'李贤成啊

刚才因为是战斗途中，没有时间欣赏，但看到这样的情景，让我深受感动。

突然我想到了什么，我转向郑熙媛。

但是见到李贤成的郑熙媛似乎没有任何兴趣。

出现异常征兆的反而是另一边。

"世仁，你健身过吗？"

"嗯？你怎么知道的？"

"盾牌动作非常明显。 你拿盾的时候用的是宽深蹲姿势，在防御上半身的时候用的是锤式弯举姿势，对吗？"

"呜哇啊，你认出来了！"

【登场人物'李贤成'对化身'京世仁'怀有好感。】

这个没关系吗？

我又回头看了看郑熙媛那边。

"为什么？"

"没什么。"

也是，仔细想想，郑熙媛和李贤成不一定非要在一起。

因为第41轮和第3轮不一样。

但即便如此，这......

【一个喜欢变换性别的星座露出有趣的表情。】

我决定还是不费心了。

现在首先要确认的是新发来的任务。

+

<主线任务#4-旗帜争夺战>

类别：主要

难度：C+

完成条件：(内容较多， 窗口已隐藏)

限制时间：14 天

奖励：2000硬币

失败时：???

+

果然，第四个任务是和正片一样的"旗帜争夺战"。

按下完成条件后，会出现一个带有长消息的窗口，就像正篇中一样。

+

【完成条件】

  1. 每个站的代表都有"旗帜"。

  2. 旗帜只有车站"代表"才能持有。

  3. 在夺取其他集团"旗帜"时，可以夺取该集团所有的"车站"。 相反，当"旗帜"被夺走时，所拥有的所有车站都会被夺走。

  4. 即使没有夺取旗帜，杀死车站代表后也会被认为夺取了旗帜，在这种情况下，将获得额外的分数。

+

果然会这样。

这次又增加了新的规则。

即使不抢夺旗帜，杀掉代表就可以了，甚至可以加分。

在"影院副本"任务中也能感受到，在第41轮的任务中，有一种奇妙的违和感。

就像有人读完了全读视正篇后，根据自己的口味改良了任务一样。

·····我想这是想太多了。

+

  1. 车站外部分地点存在隐藏"白旗"。

+

第二条新规则。

车站外的其他地方也藏着"白色旗帜"。

可能是这个任务的隐藏部分。

+

  1. 必须在14天内获得'目标站'的旗帜。如果未达成这点，团队的所有成员都会死亡。

  2. 你的"目标站"当前不存在。你无法执行该任务。

+

最后一个任务条件和我记得的差不多。

【目前，你是没有所属车站的'流浪者'。】

【你不能参加'旗帜争夺战'。】

【隐藏任务已被强加。】

我查看了随后出现的"隐藏任务"窗口。

+

<隐藏任务-流浪者>

类别：隐藏

难度：C

完成条件：进入新集团或杀死集团的代表，夺走旗帜。

限制时间：3 天

奖励：获得"旗帜争夺战"任务参战权。

失败时：? ?

+

流浪者任务。

可以说是"败者复活战"任务。

我把丹数叔叔、京世仁和郑熙媛叫来，开始谈话。

"大家都确认过任务了吧？"

"是的。"

"目前，有三种可能的选择。第一个是去别的车站，让代表接受我们。"

总之，只要找到"集团"，就能摆脱"流浪者"的惩罚，任务完成本身并不难。

京世仁摇了摇头。

"如果是这样，我就直接加入忠武路集团了。第二个呢？"

"第二是袭击其他车站，抢夺旗帜。第三是······"

"寻找隐藏的白色旗帜，对吧？"

"是的，没错。"

"那一定是第三个吧？通常第三个才是正确答案。"

确实如此。在"金独子"的全读视中。

暗中观察的京世仁小声问道。

"你知道隐藏的白色旗帜的位置吗？就像金独子知道隐藏的'绿色区域'位置一样。"

如果事情能这么顺利就好了。

"不，我不知道。"

"嗯？那怎么办？"

"所以要抢夺其他车站的旗帜。"

归根结底，"旗帜争夺战"是互相争夺旗帜的战斗。

对抗是不可避免的，我们必须摧毁其他的车站。

只有这样，隐藏任务'王者之路'才会开启，然后才能获得"绝对王座"决定战的挑战权。

丹数叔叔用阴沉的声音开口了。

"这将是一场艰难的战斗。虽然只要能夺走旗帜就好了，但事情不会那么容易。"

旗帜被夺走的车站将被剥夺车站支配权。

就连忠武路站都为此发生了争执，别说其他车站了。

郑熙媛也补充了一句。

"反正无论如何要杀掉的话，杀掉不认识的人不会那么内疚。"

丹数大叔和京世仁都被这冷漠的语气中蕴含的杀意吓了一跳。

我安慰那两个人说。

"如果可能的话，最好不要杀人，但难免会出现一些无法避免的情况。大家一定要下定决心。"

想到今后的战斗，我们在忠武路站经历的战斗只不过是儿戏而已。

我们的敌人比叶贤宇更凶恶，比韩秀英更不体贴，比孔弼斗更残酷。

仅看不久前遇到的"误读协"就知道了。

他们是一群执着的、不计其数的不遵循社会常识的犯罪分子。

【星座'断尾蛇'正在对你咆哮。】

这是一个令人恐惧的间接信息。

这些都是误读协的星座。

【星座'瘟疫之鼠'告诉你要尽可能地挣扎。】

我记得上次阿斯莫德把那只老鼠踢了出去，但祂好像又爬进来了。

【星座'断尾蛇'警告说这四个家伙就算拼了命也连星星的皮都碰不到。】

这是一场不公平的战斗。

误读协的家伙们可以在频道转播中随时监视我们，并向他们的化身泄露情报。

我想了一会儿，用"鬼怪通信"叫来了鼻荆。

-怎么了？有什么事。

'我有件事想请你帮忙。'

我列出了迄今为止我发现的"误读协"星座的名单，并问鼻荆能否阻止他们进入频道。

-你疯了吗？那些可是天上的星座们！区区一个化身……

'请听我说。据我所知，最近频道财政恶化了很多。对吗？'

鼻荆对于突如其来的财务话题一时间说不出话来。

-放肆的家伙。这不关你的事。

'这都是因为赞助。与之前相比，赞助金额恐怕已经大幅减少。是吧？'

-是减少了，但我也在寻找原因。

频道财政恶化是因为使徒们只谈论过滤的对话，导致星座们离开。

愚蠢的鼻荆似乎还不知道那个事实。

'你看过我提到的星座列表了吗？这些家伙可能没有任何赞助记录。'

-他们确实是没有赞助记录……

果然。

'是不是应该开始管理频道观众质量呢？'

-嗯嗯。虽然这话没有错，但是以没有赞助明细为由进行制裁，这有点...

'当然不能因为这样的理由进行制裁。过，我担心的是……有传言说管理局通道最近被打通了。'

-这又是从哪里听说的？

有的事实，你不必在任何地方听到就能知道。

'我认为他们不是通过正常渠道进来的。'
