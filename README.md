## 📝 剧情简介

故事讲述了来自另一个世界线的《全知读者视角》原作者**李鹤翾**。他与其他《全知读者视角》的读者们一同穿越到了《在灭亡的世界中存活的三种方法》（简称“灭活法”）之中。

然而，由于诸多读者的介入以及强大的“金独子集团”的横加干涉，导致本就艰难的任务系统彻底陷入了混乱与未知之中……
